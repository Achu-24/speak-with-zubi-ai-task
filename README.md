# 🍦 Ice Cream AI Friend

An interactive voice-based AI application designed for children.  
The AI acts as a friendly ice cream buddy that talks, listens, and continues a fun conversation for about a minute using speech recognition and speech synthesis.

The goal of this project is to simulate a natural, kid-friendly voice interaction experience using a full-stack setup.

---

## 🚀 What This Project Does

- Starts a fun AI conversation about ice cream
- Listens to the child's voice response
- Responds with cheerful, short, and engaging replies
- Continues the conversation in multiple stages
- Uses text-to-speech for AI responses
- Uses speech recognition for child input
- Maintains conversation flow without external AI dependency

The conversation includes:
- Choosing ice cream flavor  
- Selecting cone or cup  
- Adding toppings  
- Designing an ice cream shop  
- Naming the shop

## 💡 How It Works

1. User clicks **Start AI**
2. AI speaks a question
3. User responds via microphone
4. Backend processes the response
5. AI replies and continues conversation
6. Conversation resets after completion
